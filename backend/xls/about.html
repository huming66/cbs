• This application is to demo how to manage and organize data (information) with complication of related technical & business context, by using multiple levels (layer) & hierarchies (tree) interaction
    • multiple hierarchies: to organize the complicate relations
    • multiple levels: to decouple the complication
  • Taking transmission cost estimate as example application:
    • organize technical & cost details of large number of transmission components by using detail hierarchies
    • components => facility => project => scenario (plan)
    • to support transparent, consistent and efficient transmission project cost estimation and related communication
  • This simple concept (multiple levels & hierarchies) should be applicable to handle similar issues involving data (information) with complication
